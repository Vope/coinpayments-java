Usage
--------

Download [the latest release JAR][1] or grab via Maven:
```xml
<dependency>
  <groupId>org.brunocvcunha.coinpayments</groupId>
  <artifactId>coinpayments-java</artifactId>
  <version>1.0</version>
</dependency>
```
or Gradle:
```groovy
compile 'org.brunocvcunha.coinpayments:coinpayments-java:1.0'
```
